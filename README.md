## âœ¨ What is Slasher?

Slasher is a browser extension that brings the power of **slash commands** to any website. Type `/` in any text field to trigger custom commands that help you work faster and smarter.

### Key Features

- ğŸŒ **Works Everywhere** - Use your text commands on any website - GitHub, Slack, Twitter, and more.
- âš¡ **Lightning Fast** - Type `/` and instantly insert your saved text snippets. No lag, no waiting.
- ğŸ¨ **Fully Customizable** - Create your own commands
- ğŸ”’ **Privacy First** - Your snippets stay on your device. No cloud storage, no tracking and no account needed.

## ğŸ› ï¸ Development

Built with modern web technologies:

- [WXT](https://wxt.dev/) - Web extension framework
- [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Bun](https://bun.com/)

### Getting Started

```bash
# Install dependencies
bun install

# Development mode
bun run dev              # Chrome
bun run dev:firefox      # Firefox

# Build for production
bun run build            # Chrome
bun run build:firefox    # Firefox

# Run tests
bun run test
```
